# American_Binomial_Option_Pricing
The binomial method is a discretized version of Black-Scholes with n stock jumps, for some natural number N. Thus, as N approaches infinity, the model price approaches Black-Scholes under European assumptions. In this notebook, we implement the American version of the binomial method by allowing for early excersize at each step. To do this, we compare the value of continuing to hold on to the option and the current value of the option and compute the maximum. After implementing a modified version of Black-Scholes to allow for nonzero dividends, we show by comparing that, while early excersize is not optimal with zero dividends, it is for positive dividends. We then look at historic market data and compare the American binomial method to the price of historic options after calculating implied volutility.
